# Trinayani Medical - Portfolio Website

A portfolio website for Trinayani Medical, a business partner of GE Healthcare and Carl Zeiss, providing sales and service for medical equipment.

## Features

- Showcase of medical equipment (CT, MRI, Cathlab, Ultrasound from GE Healthcare)
- Eye checkup equipment portfolio (Biometry, Perimetry, Surgical Microscopes from Carl Zeiss)
- Company information and services
- Contact information
- Responsive design for all devices

## Technologies Used

- Next.js 15
- TypeScript
- Tailwind CSS
- React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website in your browser.
